Contenido del Curso

Planificación detallada de las 6 semanas de formación

Semana 1

Fundamentos de Python y Lectura de Datos

S/ 167
Hora 1: Introducción a Python, estructuras básicas
Hora 2: Lectura de CSV con Pandas
Tarea: Leer archivo y responder preguntas básicas
Se carga y visualiza cada fila que representa un segundo
Semana 2

Estadística Básica para Datos Operacionales

S/ 167
Hora 1: Media, mediana, desviación estándar
Hora 2: Outliers y agrupación por turno/unidad
Tarea: Calcular estadísticas clave
Se analiza cómo varían las variables segundo a segundo
Semana 3

Visualización de Datos

S/ 167
Hora 1: Gráficos de líneas, barras, dispersión
Hora 2: Histogramas y análisis visual
Tarea: Crear 3 gráficos de variables clave
Se grafican curvas con datos por segundo: voltaje, temperatura
Semana 4

Análisis de Series Temporales

S/ 170
Hora 1: Indexación temporal, resampling
Hora 2: Rolling mean y análisis de tendencia
Tarea: Detectar comportamientos anómalos con rolling mean
Se agrupa la data capturada por segundo para analizar por minuto, hora, etc.
Semana 5

Modelos Predictivos Básicos

S/ 167
Hora 1: Introducción a ML + Regresión
Hora 2: Árbol de decisión + separación de datos
Tarea: Modelo de predicción de variable continua
Se usan datos por segundo para entrenar modelos simples
Semana 6

Proyecto Final + Certificación

S/ 162
Hora 1: Desarrollo de mini proyecto
Hora 2: Presentación y retroalimentación
Entregable: Entrega del análisis final + certificado
Aplicación completa del curso con interpretación de datos en tiempo real
